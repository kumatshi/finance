# Финансовый калькулятор v1.0

## 1. Валидные данные для позитивного тестирования

### Кредитный калькулятор:
**Стандартный сценарий:**
- Сумма: 100000
- Срок: 12 месяцев
- Ставка: 10%
- Ожидаемый платеж: 8791.59 руб

**Минимальные значения:**
- Сумма: 1000
- Срок: 1 месяц
- Ставка: 0.1%
- Ожидаемый платеж: 1000.08 руб

**Максимальные значения:**
- Сумма: 10000000
- Срок: 360 месяцев
- Ставка: 99.9%
- Ожидаемый платеж: ≈832500.00 руб

### Конвертер валют:
**USD → RUB:**
- Сумма: 100 USD
- Ожидаемый результат: 9000.00 RUB

**EUR → RUB:**
- Сумма: 100 EUR
- Ожидаемый результат: 9850.00 RUB

**EUR → USD:**
- Сумма: 100 EUR
- Ожидаемый результат: 109.00 USD

### Калькулятор вкладов:

- Сумма: 100000
- Срок: 12 месяцев
- Ставка: 8%

## 2. Невалидные данные для негативного тестирования

### Граничные значения:
- Сумма кредита: 999 (ниже минимума)
- Сумма кредита: 10000001 (выше максимума)
- Срок кредита: 0 (ниже минимума)
- Срок кредита: 361 (выше максимума)
- Процентная ставка: 0 (ниже минимума)
- Процентная ставка: 100 (выше максимума)

### Некорректные форматы:
- Текст: "abc", "test123"
- Отрицательные числа: -1000, -50, -1
- Дробные числа для срока: 12.5, 6.7
- Пустые значения: "", " "
- Специальные символы: "!@#$", "100$"

### Некорректные валюты:
- Несуществующие коды: "GBP", "JPY", "CNY"
- Строчные буквы: "rub", "usd", "eur" (должны обрабатываться)
- Нестандартные форматы: "RUB ", " USD", "EU R"
